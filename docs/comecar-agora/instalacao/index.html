Instalação

  • Web Components
  • React

Suporte: HTML (Vue.js suportado mas não testado).

Via CDN​

Adicione as seguintes declarações à tag head de seu projeto.

<html>
  <head>
    <script type="module" src="https://cdn.jsdelivr.net/npm/@copperui/core/dist/copperui/copperui.esm.js"></script>
    <script nomodule src="https://cdn.jsdelivr.net/npm/@copperui/core/dist/copperui/copperui.js"></script>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@copperui/core/dist/copperui/copperui.css" />
  </head>
  <body>
    <!-- ... -->
  </body>
</html>

Já está pronto para o uso!

Via NPM​

Instale utilizando o gerenciador de pacotes de sua preferência.

  • npm
  • yarn
npm install @copperui/core
yarn add @copperui/core

Modo 1: JavaScript​

// Importar a lib de Web Components
import { defineCustomElements } from '@copperui/core/lib/loader';
import '@copperui/core/dist/copperui/copperui.css';
// ...
// Registrar os Web Components
defineCustomElements(window);

Modo 2: HTML​

cp node_modules/@copperui/core/dist/copperui/* {ASSETS_PATH}
<html>
  <head>
    <script type="module" src="{ASSETS_PATH}/copperui.esm.js"></script>
    <script nomodule src="{ASSETS_PATH}/copperui.js"></script>
    <link rel="stylesheet" href="{ASSETS_PATH}/copperui.css" />
  </head>
  <body>
    <!-- ... -->
  </body>
</html>

Via NPM​

  • npm
  • yarn
npm install @copperui/react
yarn add @copperui/react
