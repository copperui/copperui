# Introdução

O copperui é um projeto _open source_ desenhado para proporcionar um conjuto de ferramentas para desenvolver aplicações _Web_ modernas que sigam o [padrão de interface do Governo Federal do Brasil][dsgov].

O copperui é baseado na tecnologia de [_Web Components_][webcomponents], suportado pela maiora das plataformas populares como pode ser consultado em https://custom-elements-everywhere.com/. Um pacote de integração ao [React][react] também é fornecido.

## Começando

Crie um novo projeto em sua plataforma de preferência ou continue em um projeto já existente. Siga para a [Instalação](./instalacao).
